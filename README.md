Sistema de Control de Tiempo de Carga

Este proyecto consiste en el diseño e implementación de un sistema para el registro, control y análisis de los tiempos de carga de camiones en una operación logística de distribución de productos perecederos.

La solución surge a partir de una necesidad real dentro del entorno empresarial, donde es fundamental medir la eficiencia de los procesos de carga para garantizar la calidad del producto, optimizar la operación y respaldar decisiones estratégicas relacionadas con la expansión de la red de distribución.

El sistema permite registrar eventos clave del proceso logístico, tales como inicio y finalización de carga, salida y llegada de unidades, así como la asociación de cada evento con camiones, rutas y centros de distribución. Estos datos son almacenados de forma estructurada para posteriormente generar indicadores de desempeño, como porcentajes de cumplimiento de tiempos, análisis por sitio, por camión y por ruta.

El propósito principal del proyecto es transformar datos operativos en información útil para la toma de decisiones, alineándose con principios de control de procesos y mejora continua.

Este desarrollo se realiza como parte de la asignatura de Bases de Datos I de la Universidad Nacional Autónoma de Honduras (UNAH), integrando conceptos de modelado de datos, normalización y diseño de sistemas de información aplicados a un caso real.
