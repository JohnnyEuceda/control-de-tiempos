Sistema de Control de Tiempos de Carga

Este proyecto consiste en desarrollar un sistema para registrar y controlar los tiempos de carga de camiones en un proceso de distribución de productos perecederos.

La idea nace de una necesidad real: poder medir qué tan eficiente es el proceso de carga y usar esa información para mejorar la operación y tomar decisiones, como por ejemplo evaluar si es posible expandir la distribución.

El sistema permite registrar eventos como inicio y fin de carga, salidas y llegadas, y relacionarlos con camiones, rutas y distribuidoras. Con estos datos se pueden generar indicadores como porcentaje de cumplimiento de tiempos y análisis por sitio, camión o ruta.

El objetivo principal es convertir datos operativos en información útil para la toma de decisiones.

Este proyecto se desarrolla como parte de la clase de Bases de Datos I de la Universidad Nacional Autónoma de Honduras (UNAH), aplicando conceptos vistos en clase a un caso práctico.
